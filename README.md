# OAuth Authentication Project 

A simple *OAuth 2.0 authentication system* built with *Node.js*, allowing users to log in securely using third-party providers like **Google** or *GitHub* without sharing passwords.

#Features

* OAuth 2.0 based login
* Secure token authentication
* Third-party provider integration
* Clean Node.js backend setup

# Tech Stack

* Node.js
* Express.js
* Passport.js
* Passport-google-oauth20

#Run Project

```bash
npm install
npm start
```
